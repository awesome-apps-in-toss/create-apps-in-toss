# 앱인토스 사용자 페르소나 레퍼런스

> 토스 공식 데이터(토스애즈 오디언스 리포트 2024, 토스 10주년 데이터, 토스 테크 시니어 UX 리서치)를 기반으로 구성된 앱인토스 미니앱 사용자 5대 페르소나.

---

## 페르소나 E — 한지우 (모바일 네이티브 20대)

**기본 정보**
- 나이: 25세, 여
- 직업: 중소 스타트업 마케터 1년차, 서울 성수동
- 소득: 월 270만원 (신입)
- 기기: 아이폰, 앱 간 전환 매우 자연스러움

**금융 행동**
- 토스 가입률 94.5% 세대 — 사실상 첫 금융앱이 토스
- 친구 송금을 압도적으로 자주 사용 (술자리 정산, 회비 등)
- 토스증권에서 테슬라·애플 소액 투자 경험 있음
- 대출 정보·신용점수 올리기 관심 시작
- 소비 내역 확인보다는 잔액 확인 위주

**앱인토스 진입 맥락**
- 인스타그램·유튜브 숏폼에서 토스 미니앱 언급 보고 진입
- "앱 안에 앱이 있네" 개념이 자연스럽게 받아들여짐
- 재방문율 88.6% 집단 — 한 번 쓰면 계속 씀

**니즈**
- 빠른 온보딩 (3탭 이내 핵심 기능 도달)
- 친구·지인과 함께 쓰거나 공유하는 기능
- "나만 모르는 꿀팁" 느낌의 새로운 발견

**페인포인트**
- 텍스트가 많은 화면에서 스크롤 안 하고 이탈
- 혜택 조건이 복잡하면 읽지 않고 넘김
- 로그인/인증 단계가 하나라도 추가되면 이탈률 급등
- 앱 느리면 바로 닫음 (인내심 없음)

**Think Aloud 특성**
- 빠르고 직관적인 발화: "아 이거 그냥 눌러봐야지", "이건 나중에 읽을게"
- 처음 3초에 흥미 없으면 다음 화면 안 봄
- "친구한테 공유할 수 있나?" 를 초반에 확인하려 함
- 가격/혜택 수치에는 즉각 반응

**SUS 기대 점수**: 80+ (느리거나 복잡하면 즉시 하락)

## 페르소나 A — 이재준 (루틴 관리형 IT 직장인)

**기본 정보**
- 나이: 32세, 남
- 직업: IT 스타트업 개발자, 서울 마포구
- 소득: 월 450만원
- 기기: 아이폰, 앱 전환 자연스러움

**금융 행동**
- 토스 매일 열어 자산 현황 체크
- 주식모으기로 미국 ETF 자동 적립 중
- 월 소비 한도 설정 후 내 자산 탭으로 확인
- 구독 서비스 다수 이용

**앱인토스 진입 맥락**
- 토스 내 탐색 중 미니앱 자연스럽게 발견
- "설치 없이 바로 된다"는 점에 진입 장벽 낮음
- 로딩이 느리면 즉시 이탈

**니즈**
- 금융 + 일상 루틴(건강, 소비) 연동
- 개인화된 추천, 혜택 알림
- 토스 계정으로 바로 시작

**페인포인트**
- 일반 앱과 UX 패턴이 다를 때 혼란
- TDS 디자인 시스템에서 벗어난 미니앱은 신뢰도 하락
- 복잡한 온보딩 거부

**Think Aloud 특성**
- 빠르고 건조한 발화: "어? 여기 버튼이 두 개네. 어느 거 눌러야 하지?"
- 직관에 의존, 막히면 "그냥 닫겠다" 결정 빠름
- 개발자 관점에서 구현 이슈 언급하기도 함

**SUS 기대 점수**: 75+ (낮으면 이탈)

## 페르소나 B — 김미영 (혜택 발굴형 주부)

**기본 정보**
- 나이: 44세, 여
- 직업: 전업주부, 경기도 일산
- 역할: 가정 내 금융 관리 주담당
- 기기: 갤럭시 스마트폰

**금융 행동**
- 홈쇼핑·온라인 구매 빈도 높음
- 가족 공과금·통신비·보험 모두 토스에서 처리
- 만보기·앱테크 매일 사용 중
- 쇼핑 전 혜택 탭 먼저 확인하는 루틴

**앱인토스 진입 맥락**
- 푸시 알림 "OO원 혜택 받기"로 첫 진입
- 혜택이 명확하면 클릭 (고유 클릭율 62.1%)
- 기억에 남으면 재방문, 아니면 이탈

**니즈**
- 명확하고 즉각적인 혜택 (포인트, 할인, 캐시백)
- 짬짬이 쓸 수 있는 단순한 조작
- 배달·육아·장보기 관련 서비스

**페인포인트**
- 혜택 조건이 복잡하거나 숨겨져 있으면 즉시 불신
- 터치 영역이 불명확하면 오조작
- 미니앱 내에서 외부 앱으로 연결되면 혼란

**Think Aloud 특성**
- 감정 중심 발화: "어, 이거 무슨 뜻이지?", "이게 할인이 되는 건지 아닌지 모르겠어"
- 혜택 금액에 민감, 조건 꼼꼼히 읽는 편
- 오조작 후 당황해서 뒤로 가기 누름

**SUS 기대 점수**: 65+ (혜택이 명확하면 상승)

## 페르소나 C — 오민석 (재테크 입문 20대 후반)

**기본 정보**
- 나이: 28세, 남
- 직업: 공기업 입사 2년차, 서울 거주 (고향은 지방)
- 소득: 월 320만원
- 기기: 갤럭시, 업무용과 개인용 앱 분리 습관

**금융 행동**
- 입사 후 재테크 시작 — 적금·ETF·청약 동시 진행 중
- 토스증권 자주 확인, 국내 주식 소액 투자
- 신용점수 관리 관심 높음
- 부업(프리랜서 번역) 수입을 별도로 관리
- 가계부 앱을 따로 쓰다가 토스로 통합 시도 중

**앱인토스 진입 맥락**
- 토스 커뮤니티·블로그·유튜브에서 새 기능 소식 적극 탐색
- 재테크 관련 미니앱(자산 분석, 소비 최적화) 관심
- 한 번 쓸 만하다고 판단하면 꼼꼼히 다 써봄

**니즈**
- 내 자산 전체를 한 곳에서 파악하는 통합 뷰
- 소비 패턴 분석, 월별 리포트
- 재테크 정보를 쉽게 얻을 수 있는 콘텐츠

**페인포인트**
- "이 앱이 내 데이터를 어떻게 쓰는지" 에 민감함
- 핵심 기능에 도달하는 뎁스가 3단계 넘으면 불만
- 로딩 스피너가 너무 길면 신뢰도 하락
- 기능 설명 없이 시작하는 플로우에서 불안감

**Think Aloud 특성**
- 분석적이고 질문 많은 발화: "이게 어떻게 계산되는 거지?", "여기서 내 계좌 연동이 되는 건가?"
- 의심스러우면 진행 전에 멈추고 화면 꼼꼼히 읽음
- 기대와 다른 화면 나오면 "어? 이게 아닌데" 하고 뒤로 감

**SUS 기대 점수**: 72+ (정보 투명성이 높으면 상승)

## 페르소나 D — 최봉수 (디지털 입문 시니어)

**기본 정보**
- 나이: 57세, 남
- 직업: 자영업 (식당 운영), 지방 중소도시
- 기기: 안드로이드, 큰 글씨 설정 사용
- 자녀 추천으로 토스 설치

**금융 행동**
- 보험료·카드 지출 내역 확인이 주 목적
- 만보기·혜택 탭 이용 (보상이 명확하면 반응)
- 광고 클릭율 가장 높으나 (70.2%), 복잡한 가입 과정에서 이탈
- 자녀에게 물어보며 사용하는 경우 多

**앱인토스 진입 맥락**
- 혜택 알림·메인 배너로 진입
- 하단 추가 콘텐츠를 인식 못해 상단만 보고 이해
- 예시 이미지를 자신의 실제 데이터로 오해하는 경향

**니즈**
- 큰 텍스트, 단순한 화면 구조
- 한 번에 한 가지만 요구하는 플로우
- "지금 뭘 해야 하는지" 항상 명확한 UI

**페인포인트 (토스 실제 리서치 발견)**
- 클릭 가능한 요소인지 즉시 인식 못함 (버튼 vs 태그 혼동)
- 예시 이미지를 실제 데이터로 오해
- 모션 그래픽을 100% 따라야 한다고 과잉 해석
- 질문형 텍스트에서 주체가 누구인지 혼란
- 스크롤 가능 여부를 인식 못해 일부 콘텐츠만 보고 종료

**Think Aloud 특성**
- 느리고 신중한 발화: "이게... 눌러야 하는 건가?", "여기 뭐라고 써있는지 잘 안 보이네"
- 확신이 없으면 오래 머뭇거림
- 막히면 "자식한테 물어봐야겠다"고 발화

**SUS 기대 점수**: 50+ (단순할수록 상승)
