# 7단계: 검수 (Review Checklist)

## 목적

앱인토스 검수 기준에 맞춰 앱을 리뷰하고 QA를 수행한다.
검수는 운영/기능/디자인/보안 4단계로 진행되며, 영업일 기준 2~3일 소요.

## 검수 모드

- **light**: 핵심 항목만 빠르게 점검 (코드 레벨 자동 검증 위주)
- **full**: 공식 체크리스트 전수 검사 (게임/비게임 전체 항목)

---

## LIGHT 모드 체크리스트

코드 레벨에서 자동 검증 가능한 핵심 항목만 점검.

### 필수 설정

- [ ] granite.config.ts의 `appName`이 콘솔 등록 ID와 일치
- [ ] `brand.displayName`, `brand.primaryColor`, `brand.icon` 모두 설정됨
- [ ] `webViewProps.type`이 올바르게 설정됨 ('partner' 또는 'game')
- [ ] `navigationBar` 설정 확인 (`withBackButton`, `withHomeButton`)
- [ ] 필요한 `permissions`가 올바른 이름으로 등록됨

### TDS / 코드 품질

- [ ] `@toss/tds-mobile`, `@toss/tds-mobile-ait`, `@emotion/react` 설치됨
- [ ] `TDSMobileAITProvider`로 App 래핑됨
- [ ] TDS 컴포넌트가 최소 1개 이상 사용됨 (비게임 필수)
- [ ] TypeScript 타입 체크 통과 (`pnpm --filter @barreleye/<app-name> typecheck`)
- [ ] ESLint 통과 (`pnpm --filter @barreleye/<app-name> lint`)
- [ ] 보안 취약점 없음 (XSS, 인젝션 등)

### 핵심 UX

- [ ] Safe Area 영역 미침범
- [ ] 자사 앱/서비스 설치 유도 코드 없음
- [ ] 외부 링크가 법률 고지/공공기관 외에 없음
- [ ] iframe 미사용 (YouTube 예외)

---

## FULL 모드 체크리스트

공식 게임/비게임 출시 가이드 전수 검사.

### 공통 (게임/비게임)

#### 접속 및 기능

- [ ] 미니앱이 정상적으로 열림 (게임: 10초 이내) `[MANUAL]`
- [ ] 등록한 모든 서비스 기능이 미니앱에서 동일하게 작동 `[MANUAL]`
- [ ] 앱 스킴(`intoss://<appName>`) 연결 정상 작동 `[MANUAL]`

#### 서비스 이용

- [ ] 인터렉션 반응 **2초 이내** `[MANUAL]`
- [ ] 종료 후 재진입 시 데이터 유지 `[MANUAL]`
- [ ] 불법/선정적 콘텐츠 없음
- [ ] 비속어, 은어, 과도한 유행어 미포함
- [ ] 비정상적 네트워크/메모리 사용 없음 `[MANUAL]`
- [ ] Safe Area 영역 미침범

#### UX

- [ ] 진입 시 바텀시트 자동 오픈 없음
- [ ] 모든 화면에서 나갈 수 있는 방법 제공
- [ ] CTA 버튼 의도가 명확
- [ ] 사용자 행동 강제 유도 없음
- [ ] 자사 앱/서비스 설치 유도 없음
- [ ] 외부 링크 정상 작동 (허용 범위 내)
- [ ] 공유 기능에서 `intoss://` 스킴 사용 (private 스킴 사용 금지)

#### 권한 처리

- [ ] 권한 요청 전 사용자 동의 화면 제공
- [ ] 권한 미허용 시 나머지 기능 정상 작동

### 비게임 전용

#### 내비게이션 바

- [ ] 앱인토스 비게임 내비게이션 바 사용
- [ ] 좌측에 브랜드 로고와 미니앱 이름 표시
- [ ] 하위 화면에서 뒤로가기 버튼 동작
- [ ] 최초 화면에서 뒤로가기 시 미니앱 종료
- [ ] 우측 기능 버튼 최대 1개만 노출
- [ ] 중복 뒤로가기/홈 버튼 없음
- [ ] 더보기(...) 버튼에서 토스 공통 기능 제공

#### 기타

- [ ] 핀치줌 비활성화 (특별한 이유 없는 한)
- [ ] 라이트 모드 테마 구현
- [ ] 앱 내 기능 최소 1개 등록
- [ ] 탭바 사용 시 토스 앱과 동일한 플로팅 형태

### 게임 전용

#### 사운드

- [ ] 배경음/효과음/햅틱 구현
- [ ] 무음 모드에서 의도한 동작 유지
- [ ] 사용자가 사운드 On/Off 설정 가능
- [ ] 백그라운드 전환 시 즉시 종료, 복귀 시 정상 재생

#### 게임 기능

- [ ] 우측 상단 닫기(X) 버튼 정상 작동
- [ ] X 버튼 클릭 시 종료 확인 모달 노출
- [ ] OS 뒤로가기 제스처 사용 불가 (차단됨)
- [ ] 인게임 화면 풀스크린 구현
- [ ] 설정한 화면 방향(가로/세로) 정상 작동
- [ ] 유저 식별자 확인 및 저장
- [ ] 플레이 기록(랭킹, 레벨, 스테이지) 유지
- [ ] 종료 후 재접속 시 데이터 유지
- [ ] 주요 서비스 흐름(점수, 스테이지 클리어) 오류 없음

#### 웹보드 게임 (해당 시)

- [ ] 성인 인증 기능 연동
- [ ] 고객센터 문의 채널 제공
- [ ] 확률형 아이템 확률 정보 공개

### 수익화 기능

#### 인앱 결제 (해당 시)

- [ ] 결제 시 음악 일시 정지
- [ ] 결제창 금액과 실제 금액 일치
- [ ] 결제 완료 후 결과 정상 반영
- [ ] 결제 실패 사유 명확히 표시
- [ ] 결제 취소 정상 처리
- [ ] 취소 시 주문 화면으로 정상 복귀
- [ ] 결제 내역을 사용자가 확인 가능
- [ ] 기기 변경 시에도 구매 데이터(이용권/아이템) 유지

#### 토스페이 (해당 시)

- [ ] 결제창 금액 일치
- [ ] 결제 진행 시 음악 일시 정지
- [ ] 실패 사유 명확한 메시지 표시
- [ ] 결제 직전 취소 시 이전 화면 복귀
- [ ] 결제창 취소 시 주문 화면 이동
- [ ] 다른 결제 수단 미제공 (토스페이만)

#### 인앱 광고 (해당 시)

- [ ] 광고 재생 중 음악 일시 정지
- [ ] 광고 종료 후 음악 재재생
- [ ] 예측 불가능한 순간에 노출 없음
- [ ] 인트로/로딩/컷신/팝업 모달에 노출 금지 (게임)
- [ ] 사전 로딩 (실시간 로딩 금지)
- [ ] 광고 정상 노출 (중간 종료/튕김 없음)
- [ ] 광고 종료 후 정상 복귀
- [ ] 리워드 광고 보상 정상 지급
- [ ] 배너 광고 위치: 게임(상단/하단만), 비게임(상단/중앙/하단 허용)
- [ ] 배너 광고는 스크롤 가능 화면에만 노출 (비게임)

#### 토스 로그인 (해당 시)

- [ ] 서비스 설명을 인트로 페이지에 제공
- [ ] 약관 화면 URL 정상 노출
- [ ] 닫기 선택 시 미니앱 종료
- [ ] 자사 로그인/기타 로그인 방식 미제공
- [ ] 로그인 연결 해제 시 사용자 데이터 삭제
- [ ] 연결 해제 후 재접속 시 약관 화면 재표시

#### 공유 리워드 (해당 시)

- [ ] 공유 리워드 화면 정상 노출
- [ ] 화면 닫기 시 정상 복귀
- [ ] 친구 초대 완료 시 보상 지급

## 에이전트 기반 리뷰 방법

코드 리뷰와 체크리스트 검수를 병렬로 수행:

```
Agent 1: 코드 품질 + 설정 검증 (LIGHT 모드 항목)
  - TDS 패키지/Provider 확인
  - granite.config.ts 설정 검증
  - 권한 설정 검증
  - TypeScript/ESLint 통과

Agent 2: 체크리스트 기반 기능 검증 (FULL 모드 항목)
  - 게임/비게임별 전용 체크리스트
  - 수익화 기능 체크리스트
  - 코드 내 UX 규칙 위반 탐지
```
